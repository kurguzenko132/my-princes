# Vika Parking v11 — перенос приложения в стиль референса

## Главное

Проблема предыдущей версии: рабочий интерфейс был функциональным, но визуально не совпадал с референсом. 
Текущий v11 делает приложение намного ближе к выбранному стилю.

## Подготовленные ассеты

Папка:
`public/assets/v11`

Добавлены:
- `scene-shop-reverse-night.png` — основная реалистичная сцена практики;
- `hero-login-parking.png` — hero для входа;
- `hero-dashboard-level.png` — hero карточка dashboard;
- `hero-theory-reverse.png` — теория;
- `hero-result-scene.png` — разбор попытки;
- thumbnails уровней;
- полные reference screens;
- design board.

## Переделанные экраны

1. `/login`
   - теперь в стиле референса;
   - большой cinematic hero;
   - стеклянная карточка секретного кода;
   - нижние feature pills.

2. `/dashboard`
   - приветствие;
   - статистика;
   - большая карточка “Продолжить уровень”;
   - карточки уровней с реалистичными превью.

3. `/learn`
   - карточки уроков в новом стиле.

4. `/learn/[lessonId]`
   - теория теперь визуально как отдельный premium lesson screen;
   - большая обучающая сцена;
   - карточки шагов.

5. `/simulator/[levelId]`
   - полностью новый premium gameplay screen;
   - реалистичная ночная сцена;
   - шапка, статистика, stepper, hint card;
   - control dock как в референсе;
   - мотивационная плашка.

6. `/results/[attemptId]`
   - новый экран разбора попытки;
   - большой score;
   - scene preview;
   - “что получилось / что улучшить”;
   - действия.
