# Vika Parking v10 — управление, камера, звуки, инструктор

Сделано:

1. Виртуальный руль:
   - вместо статичного кружка теперь есть реальный touch wheel;
   - перетаскивание по рулю задаёт угол поворота;
   - стрелки по бокам остались для простого управления.

2. Физика:
   - `CarInput` получил `virtualSteering`;
   - `updateCarPhysics()` использует прямой угол руля от виртуального колеса.

3. Плавная камера:
   - сцена теперь мягко следует за машиной;
   - это делает практику ближе к игровому тренажёру.

4. Режим инструктора:
   - на идеальной траектории появились контрольные точки;
   - Вика видит последовательность движения.

5. Звуки:
   - добавлен WebAudio sound engine без mp3-файлов;
   - есть звуки клика, передачи, предупреждения и успеха;
   - настройка “Звуки” отключает их через localStorage.

6. Визуал:
   - HUD и управление стали ближе к дизайн-референсу;
   - игровой экран стал более цельным.
